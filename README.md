# WebApp de Chat - projeto acadêmico

Aplicativo de chat em tempo real com autenticação, salas privadas por código, entre outros.

---

## - Stack
- **Next.js 14** (App Router)
- **Supabase** (Auth + Realtime + PostgreSQL)
- **Tailwind CSS**
- **Vercel** (Deploy)

---

## - Funcionalidades

- + Login/Signup com email e senha
- + Entrar/Criar sala por código
- + Chat em tempo real 
- + Tema claro/escuro
- + Histórico de mensagens
- + Menu de configurações
- + Logout
